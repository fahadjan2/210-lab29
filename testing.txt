As the function is a display function, it will only have the parameter of the map being displayed, it will then take the parameter and 
process every list inside it, and for each list inside, it will correctly display what is needed to be displayed. Ie for menu items, it will
display the current menu items, for employees, will display every name working in the restaurant, and for purchases, will sum up all the purchases
and output the sum. Finally, it will return after everything is displayed.
